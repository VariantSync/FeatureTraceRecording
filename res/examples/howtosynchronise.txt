HOW TO SYNCHRONISE
Assume we edited a variant and recorded traces.
Depending on the type of edit and the traces we have to synchronise in different ways.
So what to do upon:

- INSERT:
    For all other variants v:
    If eval(config(v), new trace) equals
        false -> Do nothing
        true -> insert the new code

- DELETE:
    For all other variants v:
    If eval(config(v), new trace) equals
        false -> delete the artefact here too
        true -> do nothing (the artefact was present here before and after the edit)

- MOVE:
    For all other variants v:
    If eval(config(v), old trace) equals
        false -> If eval(config(v), new trace) then (insert artefact at the new location) else (do nothing)
        true -> If eval(config(v), new trace) then (move artefact) else (delete the artefact)

- UPDATE:
    For all other variants v:
    If eval(config(v), old trace) equals
        false -> If eval(config(v), new trace) then (insert artefact) else (do nothing)
        true -> If eval(config(v), new trace) then (update artefact) else (do nothing) // (or better delete the artefact?)
